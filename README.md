Union Database is "File Flat" database which is based on PHP and JSON.

Union Database allows the following:
REST API with JSON
Create CRUD with JSON
Add folder
Add files
Delete Files
Delete Folder
View Folders
View files
Rename folder
Rename files
Upload folder
Upload file
Export folder
Export file
Copy files
Move files
Edit files
